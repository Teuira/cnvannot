Our goals:

Implement an Easy-to-Use Open Source software for CNV interpretation. The software aggregates CNV knowledge from well-known data sources and new AI tools like X-CNV. We implemented the solution in python3, focusing on reducing the use of third-party tools. We tried to make the code as clean as possible and easy to maintain. Our work on the GUI draws inspiration from existing tools (especially X-CNV).
